<h2 id="a">A</h2>
<h3 id="°-apis">° APIS:</h3>
<p><em>DEFINICION:</em> Son conjuntos de reglas y protocolos que permiten que diferentes aplicaciones se comuniquen entre si. Facilitan el intercambio de datos y funcionalidades entre sistemas.</p>
<p><a href="#indice">volver al índice</a></p>
<h2 id="c">C</h2>
<h3 id="°-ciberseguridad">° CIBERSEGURIDAD:</h3>
<p><em>DEFINICION:</em> Área de la informática encargada de proteger sistemas, redes y datos. Busca prevenir accesos no autorizados y ataques informáticos. Incluye medidas como antivirus, firewalls y cifrado.</p>
<h3 id="°-codigo-embebido">° CODIGO EMBEBIDO:</h3>
<p><em>DEFINICION:</em> Software diseñado para ejecutarse dentro de dispositivos específicos. Se encuentra en electrodomésticos, autos y dispositivos electrónicos. Controla funciones concretas del hardware.</p>
<p><a href="#indice">volver al índice</a></p>
<h2 id="d">D</h2>
<h3 id="°-devops">° DevOps:</h3>
<p><em>DEFINICIÓN:</em> Conjunto de practicas que integran desarrollo y operaciones. Busca mejorar la colaboración entre equipos de software. Acelera la entrega de aplicaciones de forma continua.</p>
<h3 id="°-DROCKER">° DROCKER:</h3>
<p><em>DEFINICIÓN:</em> Plataforma que permite crear y ejecutar aplicaciones en contenedores. Los contenedores incluyen todo lo necesario para funcionar.</p>
<p><a href="#indice">volver al índice</a></p>
<h2 id="f">F</h2>
<h3 id="°-firmware">° FIRMWARE:</h3>
<p><em>DEFINICIÓN:</em> Software básico que controla el funcionamiento del hardware. Se almacena en memorias no volátiles del dispositivo. Permite la comunicación entre el software y el hardware.</p>
<p><a href="#indice">volver al índice</a></p>
<h2 id="h">H</h2>
<h3 id="°-hardware">° HARDWARE:</h3>
<p><em>DEFINICIÓN:</em> Conjunto de componentes físicos de un sistema informático. Incluye CPU, memoria, disco duro y periféricos. Es la parte tangible de una computadora.</p>
<p><a href="#indice">volver al índice</a></p>
<h2 id="m">M</h2>
<h3 id="°-microservicios">° MICROSERVICIOS:</h3>
<p><em>DEFINICIÓN:</em> Arquitectura de software basadas en servicios pequeños e independientes. Cada servicio cumple una función especifica se comunican entre si por medio de APIS.</p>
<h3 id="°-miniaturización">° MINIATURIZACIÓN:</h3>
<p><em>DEFINICIÓN:</em> Proceso de reducción del tamaño de componentes electrónicos. Permite crear dispositivos mas pequeños y potentes. Es clave en la evolución tecnológica.</p>
<p><a href="#indice">volver al índice</a></p>
<h2 id="pag">PAG</h2>
<h3 id="°-phishing">° PHISHING:</h3>
<p><em>DEFINICION:</em> Técnica de fraude para robar información personal. Se realiza mediante de correos o mensajes falsos. Suplantan la identidad de empresas o personas.</p>
<p><a href="#indice">volver al índice</a></p>
<h2 id="s">S</h2>
<h3 id="°-sistema-operativo">° SISTEMA OPERATIVO:</h3>
<p><em>DEFINICIÓN:</em> Software principal que gestiona el hardware del equipo. Permite la interacción entre el usuario y la computadora. Administra memoria, procesos y dispositivos.</p>
<h3 id="°-software">° SOFTWARE</h3>
<p>+DEFINICION:* Conjunto de programas y aplicaciones informáticas. Indica las instrucciones que indica el hardware. Puede ser de sistema o de aplicación. Incluye desde sistemas operativos hasta aplicaciones.</p>
<p><a href="#indice">volver al índice</a></p>
<h2 id="t">T</h2>
<h3 id="°-transistores">° TRANSISTORES:</h3>
<p><em>DEFINICIÓN:</em> Componentes electrónicos que controlan el flujo de corriente. Son la base de los circuitos integrados. Funcionan como interruptores o amplificadores.</p>
<p><a href="#indice">volver al índice</a></p>
<h2 id="v">V</h2>
<h3 id="°-virtualización">° VIRTUALIZACIÓN:</h3>
<p><em>DEFINICIÓN:</em> Tecnología que permite ejecutar varios sistemas en un solo equipo. Utiliza maquinas virtuales o contenedores. Optimiza el uso de hardware. Es muy utilizada en servidores y centros de datos.</p>
<p><a href="#indice">volver al índice</a></p>
<h2 id="w-1">W</h2>
<h3 id="°-web">° WEB</h3>
<p><em>DEFINICIÓ:</em> Conjunto de paginas y servicios accesibles por internet funciona mediante navegadores web. Utiliza tecnología como HTML, CSS y JavaScript. Permite compartir información globalmente</p>
